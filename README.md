Exploratory Data Analysis & Business Intelligence Project
Project Overview

This project analyzes retail sales data to uncover patterns, trends, and relationships. The objective is to gain insights from data using SQL queries and Python EDA, and to present them via a dashboard mock-up.

Dataset

File: superstore.csv

Contents: Order IDs, dates, customer info, product categories, sales, quantity, and profit

Purpose: Analyze sales performance, top products, categories, and geographic trends

Tools & Technologies

Python: pandas, matplotlib, seaborn

SQL: SQLite

IDE: VS Code

Dashboard: PowerPoint

Steps Performed

Data Loading: Imported CSV into SQLite database for querying

Exploratory Data Analysis:

Univariate analysis: distributions of sales, quantity, profit

Multivariate analysis: relationships between sales, category, city

SQL Queries:

Top 5 products by revenue

Monthly sales trend

Sales by category

Top cities by sales

Average order value

Total profit by category

Highest quantity products

Sales in a specific year

Sales by city & category

Highest profit products

Orders count by category

Revenue per city

Dashboard Mock-up: 9-slide PPT summarizing KPIs, charts, and recommendations

Key Insights

Electronics category generates highest revenue

Profit dips observed in some months → review promotions

Hyderabad and other top cities contribute most to sales

Average order value and total orders indicate customer behavior trends
